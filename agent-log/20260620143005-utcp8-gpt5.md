### 真实相机功能（CameraService）

`src/utils/camera.ts` 实现了完整的相机服务：

- `start(container)` — 调用 `navigator.mediaDevices.getUserMedia({ video: { facingMode: 'environment' } })` 获取真实相机流
- `stop()` — 停止所有 tracks 并清理 DOM
- `capture()` — 将视频帧绘制到 `<canvas>`，返回 `data:image/jpeg` Base64 图片
- 自动处理权限失败：显示 fallback 提示

相机在「添加」页面自动启动（进入页面时），离开页面时自动停止。
