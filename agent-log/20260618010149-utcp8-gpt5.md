# 对话与行动记录

- 用户指出 Add 页识别结果拉篮把手不应要求手动拖到底，而应像带弹性的开关，拖过一定距离阈值后松手自动滑到对边。
- 检查当前实现，发现释放时使用“最近高度吸附”逻辑，所以从 expanded 下拖不足半程会回到 expanded。
- 将释放逻辑改为基于拖动方向与阈值：
  - 拖动高度增加超过 `72px`，松手自动吸附到 `expanded`。
  - 拖动高度减少超过 `72px`，松手自动吸附到 `peek`。
  - 未超过阈值则回到拖动起点状态。
