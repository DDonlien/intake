## 完成工作

### 新增交互功能
- **日期条可切换**: 点击某一天切换选中态，同步更新能量环数据、宏量营养进度、健康同步数据；支持6天历史数据（含模拟数据）
- **餐次展开/收起**: 点击早餐/午餐/晚餐/加餐切换展开状态，再次点击收起；展开内容展示该餐次内的具体食物项
- **筛选标签切换**: 点击「全部/食物/餐次/品牌/自定义/最近」切换 active 状态，并简单过滤食物列表（品牌筛选项隐藏无品牌条目）
- **自动识别开关**: 点击切换开关状态，带动画过渡
- **添加页面关闭**: 左上角 × 按钮可返回进度页
- **添加面板可上拉**: 点击 handle 或触摸上拉/下拉展开/收起面板
- **Toast 反馈**: 点击所有可交互元素时显示轻量提示
- **实时时钟**: 状态栏时间每 30 秒更新
- **动态岛动画**: 进入添加页面时动态岛缩小变暗，模拟相机视图

### 视觉增强
- **CSS Transition 系统**: 引入 `--transition-fast`/`--transition-normal`/`--transition-spring` 三档过渡曲线
- **hover/active 反馈**: 所有卡片、按钮、行项都有 hover 上浮/缩放反馈，active 按下效果
- **宏量营养进度条动画**: 切换日期时进度条平滑过渡
- **玻璃质感微调**: 增强高光层次、内阴影、边缘折射
- **能量环 hover 效果**: 悬停时增强阴影
- **目标摘要 label 对齐**: 使用 flex 布局确保 label 顶部对齐
